## 工作打卡记录工具

> 工具选项说明

```zsh
Usage: wt [options]

Options:
  -v,    --version               查看工具版本
  -H,    --hour  [number]        上班的小时
  -M,    --mine  [number]        上班的分钟
  -LH,   --leaveHoure  [number]  规定上班的小时
  -LM,   --leaveMine  [number]   规定上班的分钟
  -c,    --config                设置上班的小时和分钟
  -h, --help                     display help for command
```
